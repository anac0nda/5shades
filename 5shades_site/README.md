# SafeOps Company Website (GitHub Pages)

A modern single-page company website for **SafeOps**.

## Structure
- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.jpg` (logo provided)

## Publish via GitHub Pages
1. Create a repository (e.g., `safeops-site`)
2. Upload all files from this folder to the repository root
3. GitHub → Settings → Pages
4. Source: Deploy from a branch
5. Branch: `main` and `/ (root)`
6. Your site will be live at: `https://<username>.github.io/<repo>/`

## Customize
Search in `index.html` for “(placeholder)” and replace with:
- real email / phone
- your domain
- any specific services wording
